MODES

Static webpage featuring two game modes for playing Rock, Paper, Scissors.

JavaScript Mode: Implemented user input functionality where users can select their choice, and the computer generates a random choice to determine the winner.
Teachable Machine Integration: Utilized Google's Teachable Machine for real-time gesture recognition, enabling users to play by showing hand gestures for Rock, Paper, and Scissors, with the application recognizing the gestures and determining the outcome.
